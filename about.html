关于美食餐厅系统

连接美食与生活，让点餐更简单

项目介绍

美食餐厅系统是一个基于现代Web技术的在线餐饮服务平台，旨在为用户提供便捷的点餐体验，为餐厅提供高效的订单管理。

响应式设计

适配各种设备，手机、平板、电脑都能完美显示

快速响应

基于现代前端技术，提供流畅的用户体验

数据安全

使用Supabase提供安全可靠的数据存储服务

技术栈

前端

HTML5 + CSS3 + JavaScript

响应式设计

后端

Supabase

实时数据库

部署

Netlify

自动化部署

版本控制

Git

代码管理

系统功能

餐厅展示

展示合作餐厅信息，包括地址、联系方式等

菜品浏览

在线浏览餐厅菜单，查看菜品详情和价格

在线点餐

选择菜品、数量，生成订单

客户管理

记录客户信息，便于后续服务
